Adobe Photoshop: Complete Guide

1. Introduction to Photoshop

Adobe Photoshop is a powerful image editing software used by designers, photographers, and digital artists. It supports pixel-based and vector-based editing, and offers advanced tools for photo retouching, composition, graphic design, and more.

2. Interface Overview

  • Menu Bar: Access all commands and settings.
  • Toolbox: Contains all the tools for editing.
  • Options Bar: Settings for the currently selected tool.
  • Panels: Layers, Colors, Adjustments, History, etc.
  • Canvas: The working area where you edit images.

3. Common Tools and Their Uses

  • Move Tool (V): Move layers, selections, and guides.
  • Marquee Tools (M): Make rectangular, elliptical, single row, or column selections.
  • Lasso Tools (L): Draw freeform selections with the standard, polygonal, or magnetic lasso.
  • Quick Selection Tool / Magic Wand (W): Quickly select areas based on tone and color.
  • Crop Tool (C): Trim, straighten, and resize the canvas.
  • Eyedropper Tool (I): Pick color samples from the image.
  • Spot Healing Brush (J): Remove blemishes or unwanted areas automatically.
  • Brush Tool (B): Paint with customizable brushes and settings.
  • Clone Stamp Tool (S): Duplicate pixels from one area to another.
  • History Brush Tool (Y): Paint a snapshot of an earlier state of the image.
  • Eraser Tool (E): Erase pixels from a layer or background.
  • Gradient Tool (G): Create smooth transitions between colors.
  • Blur/Sharpen/Smudge Tools: Softens, sharpens, or smudges areas of the image.
  • Dodge/Burn/Sponge Tools (O): Lighten, darken, or change saturation in an area.
  • Pen Tool (P): Create paths and shapes with precision curves and lines.
  • Text Tool (T): Add horizontal or vertical text to images.
  • Path Selection Tool (A): Select and move paths and shapes.
  • Shape Tools (U): Draw various geometric shapes.
  • Hand Tool (H): Move around the image when zoomed in.
  • Zoom Tool (Z): Zoom in or out of the canvas.

5. Example Projects

Design: Social Media Banner

  • Create a new 1080x1080px canvas.
  • Add a background image or gradient using the Gradient Tool.
  • Insert brand elements using the Move Tool.
  • Use the Text Tool to write a headline and adjust its style from the top bar.
  • Apply layer styles (drop shadow, glow) from Layer > Layer Style.

Design: Flyer Design

  • Create an A4 document in CMYK mode.
  • Use guides (View > New Guide) to plan layout.
  • Import product or service images, and use the Brush Tool for decorative touches.
  • Add engaging text with emphasis using contrasting colors and font weights.
  • Export as PDF or high-quality JPEG for print.

Edit: Portrait Retouching

  • Use the Spot Healing Brush to clean blemishes.
  • Dodge and Burn tools to add light and shadow.
  • Use frequency separation for high-end skin smoothing.
  • Apply color correction with Curves, Selective Color, or Camera Raw.

Edit: Background Removal

  • Select subject using Select > Subject or the Quick Selection Tool.
  • Refine with Select and Mask panel.
  • Delete background or replace it with a solid color/gradient/image.

Edit: Double Exposure Effect

  • Place two images in different layers.
  • Use blend modes (Lighten, Screen) for overlapping effects.
  • Mask parts using the Brush Tool on a layer mask.
  • Add finishing touches with Gradient Maps or Color Balance.

6. Layer Masking in Photoshop

Layer masks allow you to hide or reveal parts of a layer with precision, without deleting any content. Itâ€™s a non-destructive way to make image edits.

Steps to Use Layer Masking:

  • Select the layer you want to mask.
  • Click the "Add Layer Mask" icon at the bottom of the Layers panel.
  • Select the mask thumbnail.
  • Use the Brush Tool (B) with black to hide parts of the layer, white to reveal them, and gray for partial transparency.
  • Switch between black and white with the X key.
  • To disable the mask, Shift-click the mask thumbnail. To delete it, right-click and choose "Delete Layer Mask".

Example: Create a Smooth Blended Effect

  • Place a portrait photo over a textured background.
  • Add a layer mask to the portrait layer.
  • Use a soft round brush with black to gently blend the edges into the background.
